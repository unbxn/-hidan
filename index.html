Get the Scanner Agent
The agent runs locally on your Windows PC and sends results to this dashboard

Download Agent

The Hidan Agent is a lightweight Python script that runs the full scan locally — nothing is executed remotely. After scanning it sends a secure JSON payload to your account.

Windows 10/11 · Python 3.8+ required · ~42 KB

Setup Instructions

Follow these steps to connect the agent to your account:

  1. Install Python 3.8 or newer from python.org if you don't have it
  2. Download the agent file above and place it anywhere on your PC
  3. Open Command Prompt and navigate to the folder containing the agent
  4. Run the agent with your account token:
python hidan_agent_v1.8.py --token YOUR_TOKEN_HERE
Your token: HIDAN-XXXX-XXXX-XXXX

What the Agent Scans

All scanning happens entirely on your machine. The agent never uploads file contents — only metadata (names, paths, modification times, flags).

  • ▸ Temp directories (User + System)
  • ▸ Downloads folder and Desktop
  • ▸ Windows Prefetch directory
  • ▸ Application Event Log (IDs 1000, 1001)
  • ▸ Recycle Bin and shadow copy traces
  • ▸ Full drive scan (all connected drives)
